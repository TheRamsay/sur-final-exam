# SVM a kernel

## Co umět

- Co SVM minimalizuje/maximalizuje a význam podmínek.
- Support vectors, margin, hard-margin vs slack variables.
- Kernel function: vstup, výstup, proč nahrazuje explicitní mapování do vyšší dimenze.
- Polynomial kernel, např. `K(x,z) = (x^T z + 1)^2`, a jeho rozpis na skalární součin v rozšířeném prostoru.

## Typické zadání

- Vysvětlete rovnici SVM bez slack variables.
- Kdy je vhodnější polynomial kernel oproti lineárnímu SVM?
- Nakreslete řešení SVM pro lineárně separovatelná 2D data.
